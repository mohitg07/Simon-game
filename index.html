Press A Key to Start

Simon Game Rule

1-player game

  1. Press any key to start the game. Simon will give the first signal. Repeat the signal by pressing the same color lens.
  2. Simon will duplicate the first signal and add one more signal to it. Repeat these two signals by pressing the same color lenses, in order.
  3. Simon will duplicate these first two signals and add another signal
  4. Continue playing as long as you can repeat each sequence of signals correctly.
  5. If you fail to repeat a sequence exactly, Simon responds with a different sound. This means you've lost and the sequence of signals ends.
